## 何ができるのか？

転校を考えてるあなたの通信制高校の適応度を測ってくれます。

- 今のままの高校でもいいのかな？
- 通信制高校に向いてる人ってどんな人？
- 自分は通信制高校に向いてるのかな？

など思っている人はぜひ利用してくださいね。

## インストール方法

以下のコマンドを実行してください

```bash
npm install -g correspondence_high_school
```
